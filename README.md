# Ninjo's Player Wheel

A minimal, aesthetically pleasing spinning wheel module for Foundry VTT. Perfect for randomly selecting a player, picking targets for effects, or establishing initiative in a fun way!

## Features
- **DnD Aesthetics:** Designed with a parchment-like background and thematic colors that fit right into any dark fantasy or Dungeons & Dragons game.
- **Dynamic Spinning:** Automatically adjusts the spin duration and suspense depending on whether multiple players or just one player remains on the wheel.
- **Monk's Common Display Support:** Forces the wheel display to center perfectly onscreen if the GM decides to cast it to the entire party.
- **Fair Rotation:** Includes a tracking system where chosen players are crossed out, ensuring everyone gets a turn eventually!

## Installation
You can install this module by pasting the manifest link in your Foundry VTT Add-on Modules tab:
`https://github.com/Niclasp1501/Ninjos-Player-Wheel/releases/latest/download/module.json`

## Usage
- Open the Wheel Control interface to edit names, exclude players from the spin, or reset their status.
- Once ready, the GM can hit **Spin** to randomly select an eligible player.
